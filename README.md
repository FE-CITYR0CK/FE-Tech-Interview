## 스터디 소개

<b>'FE-Tech-Interview'</b>는 **프론트엔드 개발 분야의 기술, CS 지식, 프로젝트 경험, 최신 개발 트렌드 등**의 주제로 약 1시간 동안 발표를 통해 서로 **지식을 공유**하는 스터디입니다.

### 스터디 진행 순서

다음과 같은 순서를 한 사람씩 반복합니다.

'약 10분 발표' ➜ '5분 QnA' ➜ '스터디 종료 후 발표 내용을 정리한 마크다운 파일을 깃헙 저장소에 커밋'
